**Task - Car Price Prediction using Machine Learning**

This project focuses on predicting the selling price of used cars by analyzing key vehicle attributes and applying a machine learning regression model.

**Dataset**: Used car dataset

**Key features:**
1. Present price
2. Driven kilometers
3. Fuel type
4. Transmission
5. Number of owners
6. Car age

**Target variable**:
1. Selling price

**Workflow**
1. Data loading and cleaning
2. Feature engineering (car age)
3. Exploratory Data Analysis (EDA)
4. Categorical encoding
5. Model training and testing
6. Performance evaluation

**Model Used : Linear Regression**

1. RÂ² Score
2. Mean Absolute Error (MAE)
3. Root Mean Squared Error (RMSE)
